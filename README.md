<b>Цикломатска комплексност</b>

Цикломатската комплексност ја добив со формулата бројот на ребра (Е) - бројот на јазли (N) + 2 и изнесува 7. <br>
E = 23 <br>
N = 18 <br>
E - N + 2 = 7
